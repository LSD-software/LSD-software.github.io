💾 LSD_SOFTWARE // DIGITAL_ARTIFACTS
"Beyond Digital Boundaries. Low Poly aesthetics. Retro fueled logic. Pure Indie soul."

Benvenuto nel repository ufficiale di LSD Software, un collettivo creativo con sede a Milano, Italia. Ci specializziamo nello sviluppo di videogiochi indipendenti che sfidano le convenzioni, fondendo estetiche grezze della "golden era" a 32-bit con meccaniche di gameplay profonde e ipnotiche.

🛠 SYSTEM_SPECIFICATIONS
Core Team: Tre studenti di informatica con base a Milano.

Design Philosophy: Priorità alla libertà creativa, codice pulito e storie che lasciano un impatto.

Visual Style: Texture "croccanti", ambienti low-poly e bordi seghettati.

Tech Stack: Sviluppo orientato alle performance e accessibilità universale (Free-to-Play).
